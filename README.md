# CadenceBeep

러닝 케이던스를 위한 간단한 iOS 메트로놈 앱. 목표 분당 걸음 수(SPM)를 설정하고, 비프음을 선택한 뒤, 박자에 맞춰 달리세요.

## 기능

- **SPM 조절** — +/- 버튼 또는 프리셋(150, 160, 170, 180)으로 분당 걸음 수를 100~220 범위에서 조절
- **강박** — 2박, 3박, 4박마다 강세를 넣어 리듬감 부여
- **사운드 선택** — 높은톤, 낮은톤, 클릭 세 가지 비프음 제공
- **볼륨 슬라이더** — 다른 오디오와 믹스되어 음악을 들으며 러닝 가능

## 구조

| 파일 | 역할 |
|------|------|
| `CadenceBeepApp.swift` | 앱 진입점 |
| `ContentView.swift` | 단일 화면 SwiftUI 인터페이스 |
| `BeepEngine.swift` | 오디오 엔진 — `AVAudioEngine` 기반 사인파 비프음 생성 및 정밀 타이머 |

## 요구 사항

- iOS 16+
- Xcode 15+

## 빌드

Xcode에서 `CadenceBeep.xcodeproj`를 열고 기기 또는 시뮬레이터에서 실행하세요.
